-- Rust Commands --

- Cargo new <project_name>: initializes cargo directory with cargo.toml and main.rs file

- Cargo build: Compiles current project executable in target/debug

- Cargo run: Compiles and runs current project

- Cargo check: Checks if project can compile but doesnt compile

- Cargo build --release: Compiles with optimizations executable in target/release
